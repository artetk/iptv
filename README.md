##EPG
Самый большой подбор каналов

Ссылки на телегид с максимальным набором каналов:

    http://epg.it999.ru/epg2.xml.gz — содержит информацию на прозрачном фоне с прямоугольными пиконами;
    http://epg.it999.ru/epg.xml.gz — телегид на темном фоне с квадратными пиконами;
    http://epg.it999.ru/pp.xml.gz — список, собранный специально для Perfect Player и ProgTV;
    http://epg.it999.ru/epg2.xml — упрощенный файл, содержит телепрограмму IPTV на прозрачном фоне с прямоугольными иконками;
    http://epg.it999.ru/epg.xml — упрощенный файл, отображает ТВ-гид на тёмном фоне с квадратными иконками.

Упрощенный файл не содержит gzip-сжатия. В нём программа на текущий день с минимальным количеством информации, что позволяет воспроизводиться даже на слабых устройствах.
Русскоязычные каналы

Ссылки на телегид только с каналами на русском языке:

    http://epg.it999.ru/ru2.xml.gz — содержит ТВ-программу для IPTV на прозрачном фоне с прямоугольными пиконами;
    http://epg.it999.ru/ru.xml.gz — телепрограмма с квадратными пиконами на темном фоне;
    http://epg.it999.ru/rupp.xml.gz — подборка для Perfect Player и ProgTV.
