Ссылки на телегид с максимальным набором каналов:

(42Mb)  http://epg.it999.ru/epg2.xml.gz — содержит информацию на прозрачном фоне с прямоугольными пиконами;
(42Mb)  http://epg.it999.ru/epg.xml.gz — телегид на темном фоне с квадратными пиконами;
(35Mb)  http://epg.it999.ru/pp.xml.gz — список, собранный специально для Perfect Player и ProgTV;
(10Mb)  http://epg.it999.ru/epg2.xml — упрощенный файл, содержит телепрограмму IPTV на прозрачном фоне с прямоугольными иконками;
(10Mb)  http://epg.it999.ru/epg.xml — упрощенный файл, отображает ТВ-гид на тёмном фоне с квадратными иконками.

Упрощенный файл не содержит gzip-сжатия. В нём программа на текущий день с минимальным количеством информации, что позволяет воспроизводиться даже на слабых устройствах.
